# Cron Jobs

Define scheduled functions using the `@Cron` decorator on injectable service methods.

## Defining a Cron Job

```typescript
import { Injectable } from '@nestjs/common';
import { Cron, PingbackContext } from '@usepingback/nestjs';

@Injectable()
export class CleanupService {
  @Cron('daily-cleanup', '0 3 * * *', { retries: 3, timeout: '60s' })
  async cleanup(ctx: PingbackContext) {
    const expired = await this.removeExpiredSessions();
    ctx.log('Sessions cleaned up', { removed: expired });
    return { removed: expired };
  }
}
```

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `retries` | 0–10 | 0 | Retry attempts with exponential backoff |
| `timeout` | string | "30s" | Max execution time (e.g. "30s", "5m") |
| `concurrency` | 1–10 | 1 | Max concurrent executions |

## Schedule Expressions

Standard 5-field cron expressions:

| Expression | Description |
|-----------|-------------|
| `*/15 * * * *` | Every 15 minutes |
| `0 3 * * *` | Daily at 3:00 AM UTC |
| `0 */6 * * *` | Every 6 hours |
| `0 9 * * 1` | Every Monday at 9:00 AM |
| `0 0 1 * *` | First day of every month |
