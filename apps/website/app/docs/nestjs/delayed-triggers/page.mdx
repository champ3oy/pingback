# Delayed Triggers

Trigger a task with a delay — the execution will be scheduled to run after the specified time.

## Usage

```typescript
import { Injectable } from '@nestjs/common';
import { PingbackClient } from '@usepingback/nestjs';

@Injectable()
export class NotificationService {
  constructor(private readonly pingback: PingbackClient) {}

  async scheduleReminder(userId: string) {
    // Send reminder 24 hours from now
    await this.pingback.trigger(
      "send-reminder",
      { userId },
      { delay: "1d" }
    );
  }

  async scheduleFollowUp(userId: string) {
    // Send follow-up in 30 minutes
    await this.pingback.trigger(
      "send-followup",
      { userId },
      { delay: 1800 }
    );
  }
}
```

## Delay Formats

| Format | Example | Description |
|--------|---------|-------------|
| Seconds (integer) | `900` | Delay by 900 seconds |
| Seconds | `"30s"` | Delay by 30 seconds |
| Minutes | `"15m"` | Delay by 15 minutes |
| Hours | `"2h"` | Delay by 2 hours |
| Days | `"1d"` | Delay by 1 day |
| Combined | `"1d2h30m"` | Delay by 1 day, 2 hours, 30 minutes |

Maximum delay: 30 days.
